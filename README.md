# Salary Prediction using Simple Linear Regression

This project demonstrates a basic Machine Learning workflow using **Simple Linear Regression** to predict employee salaries based on their years of experience.

It was built as one of my first Machine Learning projects using Python and scikit-learn.

---

## Project Overview

The notebook covers the complete process of building a simple regression model:

- Importing required libraries
- Loading the dataset
- Splitting data into training and testing sets
- Training a Simple Linear Regression model
- Making predictions
- Visualizing training and testing results
- Evaluating the model using common regression metrics

---

## Dataset

The project uses the **Salary_Data.csv** dataset.

Features:

- **YearsExperience** → Input feature
- **Salary** → Target variable

## How to Run

Clone the repository:

```bash
git clone https://github.com/aiotaha/salary-prediction-linear-regression.git
```

Install the required libraries:

```bash
pip install numpy pandas matplotlib scikit-learn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
linear_Regression_Salary_Data.ipynb
```
